README
======

Описание проекта
----------------

Данный проект представляет собой веб-приложение для управления учетными записями. Пользователи могут добавлять, редактировать и удалять учетные записи, а также сохранять их в локальном хранилище браузера. Приложение построено с использованием Vue 3 и Element Plus.

Структура проекта

│ \

Установка
---------

1.  Клонируйте репозиторий:
    `git clone <URL_репозитория> 2cd <имя_папки_репозитория>`
    
2.  Установите зависимости:  
    `npm install`
    
3.  Запустите проект:
    `npm run dev`
    
4.  Откройте браузер и перейдите по адресу `http://localhost:5173` (или другой, указанный в консоли).
    

Использование
-------------

*   **Добавление учетной записи**: Нажмите на кнопку `+`, чтобы добавить новую учетную запись.
*   **Редактирование учетной записи**: Введите данные в поля `Метка`, `Тип`, `Логин` и `Пароль`.
*   **Удаление учетной записи**: Нажмите на кнопку с иконкой удаления рядом с учетной записью.
*   **Удаление всех учетных записей**: Нажмите на кнопку `Удалить все`, чтобы очистить все учетные записи.

Правила валидации
-----------------

*   **Метка**: Должна быть строкой и не превышать 50 символов.
*   **Логин**: Обязателен и не должен превышать 100 символов.
*   **Пароль**: Обязателен и не должен превышать 100 символов (для локальных учетных записей).

Хранение данных
---------------

Данные учетных записей сохраняются в локальном хранилище браузера, что позволяет сохранять их между сессиями. При загрузке приложения данные загружаются из локального хранилища.

Технологии
----------

*   **Vue 3**: Основной фреймворк для создания пользовательского интерфейса.
*   **Element Plus**: Библиотека компонентов для Vue.
*   **Pinia**: Хранилище для управления состоянием приложения.
*   **TypeScript**: Язык программирования, используемый для разработки.
